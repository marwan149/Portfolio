## ✨ Features

- **Animated particle background** — 150 floating red particles rendered on a `<canvas>`
- **3D tilt effect** on project cards (pure CSS + JS, no library)
- **Glitch animation** on the hero title
- **Scroll reveal** — sections fade in as you scroll
- **Glassmorphic sticky navbar** with mobile hamburger menu
- **Responsive layout** — adapts to mobile, tablet, and desktop
- **Floating hero icon** with smooth CSS keyframe animation

## 🎨 Customization

All styling is in the `<style>` block inside `index.html`. Key design tokens:

```css
--red:       #ff0040;   /* primary accent */
--bg:        #0b0b0b;   /* page background */
--card-bg:   rgba(20,20,20,0.9);
--font-display: 'Orbitron', sans-serif;
--font-body:    'Inter', sans-serif;
```

To swap the accent color, do a find-and-replace on `#ff0040`.
